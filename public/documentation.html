This is a simple to use api that allows developers to access information about movies. This includes, information about actors , genres , movie budget , box office sales and more to come!

Endpoints:

/movies

Data returned : Data holding all information on all movies in the movie list. Sent in JSON format. "title" , "Genre" , "Director" , "imageURL" , "description" , "featured"

HTTP Method : GET

Query Parameters: none

Request Body Format : no request body needed

Response Body Format : JSON

/movies/:movie

Data returned : json object holding information on a single movie "title" , "Genre" , "Director" , "imageURL" , "description" , "featured"

HTTP Method : GET

Query Parameters: :movie

Request Body Format : no request body needed

Response Body Format : JSON

/movies/:movie/genre

Data returned : json object holding the genre of the movie "name" , "description"

HTTP Method : GET

Query Parameters: :movie

Request Body Format : no request body needed

Response Body Format : JSON

/movies/:movie/director

Data returned : json object holding data about the director "name" , "bio" , "born" "death"

HTTP Method : GET

Query Parameters: :movie

Request Body Format : no request body needed

Response Body Format : JSON

/users

Data returned : status code of 201 as well as data of the newly entered user with a success message

HTTP Method : POST

Query Parameters: none

Request Body Format: "birthday": "04-14-1996", "email": "jp_goats@yahoo.com", "password": "mike123", "username": "mike"

Response Body Format : JSON

/users/:Username

Data returned : The newly inserted data for the user in json format as well as a status code of 201

HTTP Method : PUT

Query Parameters :Username

Request Body Format : "FavoriteMovies": [], "_id": "60753df5501f6542848300ba", "username": "mike", "password": "mike456", "email": "jp_goats@yahoo.com", "birthday": "1996-04-14T07:00:00.000Z", "__v": 0

Response Body Format : JSON

/users/:Username/favorites/:MovieID

Data returned : json object containing the newly added favorite movie for the user

HTTP Method : POST

Query Parameters: Username , MovieID

Request Body Format : none

Response Body Format : JSON

/users/:Username/favorites/:MovieID

Returns updated data in json

HTTP Method : DELETE

Query Parameters: Username , MovieID

Request Body Format : no request body needed

Response Body Format : JSON

/users/:Username

Data returned : message indicating the user was successfully deleted

HTTP Method : DELETE

Query Parameters: Username

Request Body Format : not yet specified

Response Body Format : string
